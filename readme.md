Clona el repositorio, cambiar version de maven e instala las dependencias:

```bash
git clone https://github.com/leluza/AppFullStackParaPortfolioFRONT
cd frontend
nvm use 16
npm install
npm run start
abrir http://localhost:3000
y acceder a la opcion del menu "CV"
